Welcome to the Wind Turbine Airfoil Catalogue website!


This website is dedicated to give access to data from the Wind Turbine Airfoil Catalogue. This catalogue gathers numerical and experimental data obtained for a wide range of airfoils used in wind turbine applications.

For each computed airfoil, numerical results have been obtained with the incompressible Navier-Stokes solver EllipSys2D and the panel method code XFOIL (more details on these computational codes here). In most cases, these numerical results are compared to experimental measurements when available. Note that all measurements available on this site were performed at low Mach numbers so that the fluid can be considered as being nearly incompressible.

The primary goal of this catalogue was to provide this database. However, it was used to validate and evaluate the computational code EllipSys2D.
